# Project Board

Project Board is a MEAN stack web application where one register and keep the records of their projects.

---

## Table of Content

- [Features](#features)
- [Technologies Used](#technologies-used)
- [Installation](#installation)

### Features

- **User Authentication**: Secure user login and registration with JWT-based authentication.
- **Lazy-Loading**: Implement lazy-loading for authentication for loading on demand.
- **Project Management**: Add, edit, and delete projects with detailed information like name, client, and status.
- **Role-Based Access**: Different functionalities for Admin, Manager, and Employee roles.
- **Real-Time Updates**: Projects update dynamically without requiring a page refresh.
- **Responsive Design**: Fully responsive and mobile-friendly layout using Bootstrap.
- **Search and Filter**: Quickly find projects using search and advanced filtering options.

### Technologies Used

- Frontend: HTML5, CSS3, Bootstrap, Angular 16, TypeScript, Angular Material
- Backend: Node.js, Express.js, Mongoose
- Database: MongoDB
- Others: JWT for Authentication, etc.

### Clone the Repository

```bash
git clone https://github.com/Prabhav-Manas/Project-Board.git
```

#### Navigate to the Project Directory

```bash
cd Project-Board
```

#### Install Dependencies

```bash
Frontend:
cd frontend
npm install
```

```bash
Backend:
cd backend
npm install
```

## Usage

### Backend

### Frontend
